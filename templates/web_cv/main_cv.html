    Este CV

    La idea de este CV es presentar y demostrar mis conocimientos sobre programación. Para lograr esta página se utiliza HTML, CSS, Python y Django Framework entre otros conocimientos sobre servidores web y demás. El Diseño del Front-End de esta página web es colaboración de Clarence Taylor, quien presta el código HTML y CSS general para quien lo desee. La diferencia entre el proyecto de Taylor y el mío es la funcionalidad otrogada con python a través de Django. Las distintas listas de experiencia laboral, programación y educación formal forman parte de la base de datos, la cual se muestra de manera automatica gracias a python. Si deseo agregar uno o mil items a estas listas, puedo hacerlo muy rápidamente modificando la base de datos sin modificar en absoluto el codigo del Front-End. El código entero de esta página esta en el repositorio CV de Github.
